## P0 试产结论（量产配方）

- **贴脸部件（发型/胡子/衣服等）**：基准头 i2i「保持…完全不变,只改X」→ 与基准逐像素差分提取（阈值 |ΔRGB|>60）→ 部件自带对位零校准。实测非目标区误改仅 0.1%。
- **紧凑部件（眼睛/眼镜/饰品）**：可独立生成（绿幕悬空展示）+ 手工锚点校准补充。
- **绿幕抠图**：#00FF00 背景 + 色键（greenness=g−max(r,b)，hi=90/lo=20 羽化）+ 去溢色；深色发丝边缘需加强 despill + 1px 收边。
- **跨性别部件不可移植**，男女各用各自基准生成（正好契合男女分组需求）。
- 脚本归档在 `p0-archive/`（生成/抠图合成/差分统计/报告页），P2 量产脚本以此为底。

## 模型 API 要点（docs.volcengine.com/docs/82379/1541523）

- **量产必开 `output_format:'png'`**（5.0 pro/lite）——消差分提取的 JPEG 噪点。
- **组图**：`sequential_image_generation:'auto'` + `sequential_image_generation_options.max_images`（仅 5.0 **lite** / 4.5 / 4.0；pro 不支持）——单参考图一次出 ≤14 张关联变体，量产批次利器。
- **5.0 pro 独有**：2~10 参考图 + 交互编辑（坐标/框选精准改区域）。
- 尺寸：pro 方式2 总像素最低 92 万（可 1024×1024 直出省 3/4 成本）；lite/4.5 最低 368 万（组图批次用 2K）。
- 无 seed 参数（复现靠存产物）；结果 URL 24h 失效（即时下载）；watermark 默认 true 须显式关。
- ⚠ 账号「安心体验模式」有单日推理限额，量产前确认已调整（2026-07-15 曾触发 429 SetLimitExceeded 全天暂停）。
